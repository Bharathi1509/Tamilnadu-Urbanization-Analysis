# Tamil Nadu Urbanization Analysis (Power BI)

## Overview

This Power BI project analyzes **urbanization patterns in Tamil Nadu** using demographic and regional data.
The dashboard focuses on understanding how population distribution, urban growth, and regional differences
vary across districts and over time.

This is a **data visualization and insight-driven project**, not a predictive model.

---

## Objectives

- Analyze urban vs rural population distribution
- Compare district-wise urbanization levels
- Identify highly urbanized and least urbanized regions
- Present insights through interactive dashboards

---

## Dataset Description

The dataset used in this project includes attributes such as:

- District
- Total Population
- Urban Population
- Rural Population
- Urbanization Percentage
- Growth Indicators (if available)

> The dataset is assumed to be pre-cleaned before being loaded into Power BI.

---

## Key Features

### Power BI Dashboards
- District-wise urbanization comparison
- Urban vs rural population breakdown
- Percentage-based urbanization metrics
- Interactive slicers for district-level filtering
- KPI cards highlighting key statistics

### Data Modeling
- Calculated columns and measures using DAX
- Percentage calculations for urban population
- Aggregations at district and state level

---

## Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Excel / CSV (data source)
